# Konsep Bisnis — AURELUX.BEAUTY

> Dokumen ini disusun sebagai panduan bagi tim developer dalam memahami konsep bisnis, identitas brand, dan alur proses bisnis Aurelux Beauty. Tujuannya agar pengembangan website dapat dilakukan dengan pemahaman yang utuh terhadap kebutuhan dan karakter brand.

## 1. Tentang Aurelux Beauty

**AURELUX.BEAUTY** adalah brand skincare harian yang dirancang untuk menjaga kulit tetap **tenang, nyaman, dan stabil**, sehingga dapat digunakan secara konsisten tanpa rasa khawatir.

Brand ini berfokus pada **perawatan kulit sehari-hari** yang simpel dan aman, cocok untuk pengguna yang menginginkan rutinitas skincare ringan namun efektif.

## 3. Tujuan Website

Website Aurelux Beauty **bukan** website e-commerce. Website ini berfungsi sebagai:

- **Brand Awareness** — Memperkenalkan brand dan membangun kepercayaan
- **Official Presence** — Menampilkan identitas resmi perusahaan secara profesional
- **Product Showcase** — Menampilkan produk dengan visual yang menarik dan besar
- **Lead Generation** — Mengarahkan calon pelanggan ke WhatsApp untuk pembelian

### Yang Perlu Diperhatikan Developer

1. **Gambar produk harus ditampilkan besar dan menonjol** — ini permintaan spesifik dari klien
2. **Tidak ada fitur transaksi/e-commerce** — website murni untuk branding
3. **CTA (Call to Action) mengarah ke WhatsApp** — bukan keranjang belanja
4. **Admin panel diperlukan** — untuk mengelola konten website
5. **Profil perusahaan** dibuat oleh tim Kreatifindo (bukan dari klien)

## 4. Proses Bisnis (Alur Pengguna)

Berikut alur yang terjadi ketika seseorang mengunjungi website Aurelux Beauty:

### 4.1 Alur Pengunjung Website

```
Pengunjung datang ke website
        │
        ▼
Melihat halaman utama (Hero section + visual produk)
        │
        ▼
Menjelajahi informasi produk (Toner / Moisturizer)
        │
        ▼
Membaca profil perusahaan / tentang brand
        │
        ▼
Tertarik dengan produk
        │
        ▼
Klik tombol CTA → Diarahkan ke WhatsApp
        │
        ▼
Konsultasi / Pembelian via WhatsApp
```

### 4.2 Alur Admin (Pengelola Website)

```
Admin login ke admin panel
        │
        ▼
Mengelola konten website:
  - Update informasi produk
  - Update gambar/visual
  - Update profil perusahaan
  - Update kontak/link WhatsApp
        │
        ▼
Konten ter-update di website publik
```

## 6. Tone & Visual Brand

### Personality Brand
- **Calm & Soothing** — Tenang, menenangkan
- **Clean & Minimal** — Bersih, tidak berlebihan
- **Trustworthy** — Bisa dipercaya, profesional

### Panduan Visual
- Warna-warna yang **lembut dan menenangkan** (sesuai identitas brand)
- Tipografi yang **bersih dan modern**
- Foto produk ditampilkan dengan **ukuran besar dan dominan**
- Layout yang **tidak terlalu ramai**, mengutamakan whitespace
- Referensi visual final akan diberikan oleh **tim produksi Kreatifindo**

### Video Promosi
- Video promosi fokus pada **awareness** (bukan hard selling)
- Video teaser yang sudah ada bisa diintegrasikan ke website
